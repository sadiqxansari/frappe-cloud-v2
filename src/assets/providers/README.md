# Provider logos

Drop a logo here named by the provider id and `ProviderIcon.vue` renders it
everywhere automatically (monogram tile is the fallback until the file exists).

Accepted: `.svg`, `.png`, `.webp`. Square-ish, transparent-safe artwork works best.

| File | Provider |
| --- | --- |
| `aws.svg` | Amazon Web Services |
| `hetzner.svg` | Hetzner |
| `frappe.svg` | Frappe |
| `oracle.svg` | Oracle Cloud |
| `digitalocean.svg` | DigitalOcean |
